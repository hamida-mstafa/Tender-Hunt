# Tender Hunt
#### Its an online application for posting tenders , 2018/10/05
#### By *Hamida Mstafa Mohamed* *Kamadi Douglas* *James Oduor* *Stephene Otieno*

## Description
Tender hunt is an online tenders publishing website majoring in ensuring suppliers get to view all tenders before their due date is reached, and also customers get to ask for the goods and services they require in time.
